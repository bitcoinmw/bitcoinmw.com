• BitcoinMW is a fork of Bitcoin (BTC) which installs the UTXO set in a Mimblewimble based (Grin fork) application.
  • The BTC UTXO set is compressed into a binary file that is less than 1 GB.
  • This binary file is downloaded automatically via peer to peer network upon startup of the BitcoinMW full node.
  • BitcoinMW is a 100% fair launch. No ICO, premine, dev fund, dev tax, etc.
  • BitcoinMW is scarce, maintaining Bitcoin's 21m supply cap, with a tail emission of 1,000 years. (See consensus.rs)
