• BitcoinMW is a hard fork of Bitcoin (BTC) which installs the UTXO set into a new Mimblewimble based (grin fork) application.
  • The BTC UTXO set is compressed into a binary that is less than 1 GB.
  • This binary is downloaded automatically via peer to peer network upon startup of the BitcoinMW full node.
  • BitcoinMW is a 100% fair launch. No ICO, premine, dev fund, dev tax, etc.
  • BitcoinMW is scarce, maintaining Bitcoin's 21 million supply cap, but with a long tail emission of 1,000 years. (See consensus.rs)
  • Full node on Github: https://github.com/bitcoinmw/bitcoinmw
  • Wallet on Github: https://github.com/bitcoinmw/bmw_wallet
